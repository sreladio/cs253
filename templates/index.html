    Info

    Hola, me llamo Carlos Eladio Fernández Ordoñez, y este es el sitio Web que he desarrollado para el curso Web Development de Udacity.

    Está desarrollado con Python para el lado del servidor y Jinja2 como lenguaje de plantillas. Para el framework web he utilizado Webapp2, ya que permite un desarrollo a un nivel más bajo que Django.

    Puedes ver y dercargar el código en Github.

    Desarrollo

    Las funcionalidades implementadas son:

    • Registro de nuevos usuarios (Signup):
      • Validación de entrada de datos mediante expresiones regulares
      • Las contraseñas son hasheadas con bcrypt antes de ser almacenadas en la BB.DD.
      • Función "remember me". Si un usuario que ya está logeado intenta acceder a la página de registro, se le redirectiona a la página de vienvenida
    • Login:
      • Las cookies son hasheadas mediante bcryp
      • Verificación de las cookies cuando se carga una página
      • Función "remember me". Si un usuario que ya está logeado intenta acceder a la página de login, se le redirectiona a la página de vienvenida
    • Logout:
      • Borra la cookie y redirecciona a la página de registro
    • Blog:
      • New Post
      • JSON
      • Clear Memcache
